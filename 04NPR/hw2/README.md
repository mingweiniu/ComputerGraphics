# glfw_glew_glm

this is simple framework for drawing 
key board please see controls.txt

## Prerequisites
  * C++14 compiler  
  * CMake (3.5 or latter)  
  * glfw (3.2 or latter)  
  * glew (2.0 or latter)  
  * glm (0.9.8.4 or latter)  
  * pkg-config (if UNIX)  

### Build
#### Win10: 
PS\\> .\build.bat
#### Linux:
[user]$ bash build.sh
